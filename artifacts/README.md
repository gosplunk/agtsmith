# Artifact Repository

Operational outputs are stored here so documentation pages stay focused on engineering docs and runbooks.

## Layout
- `artifacts/runs/langgraph/` minimal flow run artifacts
- `artifacts/runs/agentic/` agentic loop run artifacts
- `artifacts/runs/multi_model/` multi-model run artifacts
- `artifacts/environment/` environment profiles and history snapshots
- `artifacts/knowledge/` generated skillpacks and prompt-assist knowledge artifacts
- `artifacts/sessions/` session memory state files
- `artifacts/model_eval/` model benchmarking outputs

## Notes
- This directory is intended for machine-readable outputs, reports, and execution traces.
- Web UI exposes a browser for this repository at `/artifacts`.
